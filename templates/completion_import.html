How It Works

Paste spreadsheet rows or upload a CSV to expand set SKUs into member items and roll duplicate ownership quantities together.

  • Set SKUs expand into their member item SKUs before totals are rolled up.
  • Unknown item or set SKUs are skipped, not created.
  • Duplicate rows for the same person, item, and color are summed.
  • Existing ownership rows are updated, not duplicated.
  • Quantity is added to Quantity Purchased for matching ownership rows.

Example Paste

person,sku,quantity,note
Anthony Child,1726,1,ordered from rep
Anthony Child,326,1,from set
Anthony Child,326,2,ordered again

That would roll into one direct item row and one expanded set row, with duplicate quantities summed before confirm.

Next: we’ll roll set SKUs into their member items, merge duplicates, and show unresolved rows before anything is saved.
