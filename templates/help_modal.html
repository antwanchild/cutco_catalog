{% if page == 'catalog' %}

The catalog lists every Cutco item known to the vault.

  • Search by name or SKU using the search box.
  • Filter by category or show only 🦄 unicorn items.
  • Sort any column by clicking its header.
  • Click an item name to see its full detail page.
  • Admin users can add, edit, or delete items.
{% elif page == 'sets' %}

Sets are curated groupings of Cutco items (e.g. a knife block set).

  • Select a person from the dropdown to see their personal completion percentage for each set.
  • Click a set name to see which items are owned vs. missing.
  • Sets are created automatically during a Catalog Sync, or manually with the + New Set button.
{% elif page == 'set_detail' %}

Shows which items in this set are owned and which are still missing.

  • Select a person to filter completion to their collection.
  • Click + Wishlist on any missing item to add it to that person's wishlist.
  • Click 🎁 Share Gift List (when a person is selected) to generate a shareable link showing only the missing items — great for gifting.
{% elif page == 'collection' %}

Shows everything a person owns, wants, or has sold/traded.

  • Use the checkboxes to select multiple entries, then use the bulk action bar to change their status at once.
  • Missing Items lists catalog items this person doesn't have yet — click + Add to log them.
  • Variant Gaps shows items they own but are missing specific color variants of.
  • Click 🃏 Share Card to generate a public shareable link of their collection.
{% elif page == 'sharpening' %}

Tracks when each knife was last sharpened.

  • Knives past the threshold (default 180 days) are marked overdue.
  • Cookware, bakeware, gift boxes, sheaths, gadgets, and storage are excluded because they do not have sharpenable edges.
  • Log a new sharpening event with the form — pick the knife, date, and method.
  • Admin users can send overdue reminders to Discord.
  • The threshold is configurable via the SHARPEN_THRESHOLD_DAYS environment variable.
{% elif page == 'cookware' %}

Tracks usage sessions for Cutco cookware pieces.

  • Log what you made, a rating (1–5), and optional notes.
  • Items unused past the threshold (default 60 days) are flagged as stale.
  • The threshold is configurable via COOKWARE_THRESHOLD_DAYS.
  • Admin users can send stale reminders to Discord.
{% elif page == 'tasks' %}

Log which knife you use for which kitchen task to track usage patterns.

  • Only owned knives appear in the log form.
  • The Usage Patterns table shows each knife's most-used task.
  • 10 preset tasks are included — add your own via Manage Tasks.
  • Custom tasks can be deleted if they have no logged usage.
{% elif page == 'wishlist' %}

Tracks items people want to own, optionally with a target price.

  • Set a target price on a wishlist entry — when the item's MSRP drops to or below that price, it's highlighted as a 🎯 hit.
  • Filter by person using the dropdown.
  • Click the column headers to sort by item name, SKU, or target price; clicking an active header toggles asc/desc.
  • Admin users can trigger a price check that sends Discord alerts for any hits.
{% elif page == 'matrix' %}

A cross-tab view showing who owns what at a glance.

  • Rows are items, columns are people.
  • Each cell shows ownership status color-coded by badge.
  • Click the Name or SKU header to sort the item rows; clicking the active header toggles asc/desc.
  • Useful for spotting gaps or duplicates across your group.
{% elif page == 'stats' %}

Charts and summaries of the collection.

  • Filter to a specific person to see their individual stats.
  • Stats include public items plus unicorn, rep only, Costco, and non-catalog items marked Owned.
  • Charts show owned items by category, handle color distribution, edge type breakdown, catalog coverage, and estimated collection value.
  • The Collectors table ranks people by item count and estimated value.
{% elif page == 'item_detail' %}

Full detail view for a single catalog item.

  • Left column — color variants, sets this item belongs to, and top tasks it's been used for.
  • Right column — who owns it, with quick-add buttons for people who don't.
  • Sharpening history and task log appear below if any events have been logged.
{% elif page in ['sync', 'sync_preview'] %}

Scrapes Cutco.com to discover new items and sets.

  • New items are grouped by category — check the ones you want to import.
  • New sets are listed separately — check any you want to add.
  • Items in blocked categories (e.g. Accessories) are hidden by default.
  • Only items with SKUs not already in the database are shown.
{% elif page in ['import', 'import_page'] %}

Bulk-import ownership data from a CSV or XLSX file.

  • Recommended header order: name, sku, owned, color, availability, quantity purchased, quantity given away, category, edge, is_sku_unicorn, is_variant_unicorn, is_edge_unicorn, price.
  • Core columns: name, sku, owned.
  • Optional columns: color, quantity_purchased, quantity_given_away, category, edge, is_sku_unicorn, is_variant_unicorn, is_edge_unicorn, price.
  • Unicorn flags accept truthy values like yes, x, true, and 1.
  • Items can also store Alternate SKUs on the edit form so imports can match legacy or vendor-specific model numbers.
  • Headers are matched case-insensitively, so lowercase is recommended only for consistency.
  • Legacy files may still use is_color_unicorn.
  • Owned can be a yes/no value, a collector name, or a legacy status/person value.
  • A preview is shown before any data is written — you can deselect rows.
  • Existing ownership entries are skipped (no duplicates created).
{% elif page in ['completion_import', 'completion_import_page'] %}

Paste or upload a list of SKUs to build a completion rollup for a collector.

  • Set SKUs expand into their member item SKUs before totals are rolled up.
  • Unknown item or set SKUs are skipped, not created.
  • Duplicate rows for the same person, item, and color are summed.
  • Matching ownership rows are updated by adding to Quantity Purchased.
  • Use headers person, sku, quantity, note.
  • Example: Anthony Child,1726,1,ordered from rep or Anthony Child,326,1,from set.
{% elif page in ['completion_gaps', 'completion_gaps_page'] %}

Shows the public catalog SKUs a collector still does not own in any variant, and can also export them as CSV.

  • Choose one collector or all collectors from the dropdown.
  • The report checks public catalog items only.
  • If a collector owns any variant of an item, that item is not listed as missing.
  • Use View on screen to see the missing SKUs in a table before downloading.
  • Download the result as a CSV you can keep or share.
{% elif page == 'dashboard' %}

Welcome to Cutco Vault — a self-hosted tracker for Cutco knife and kitchen product collections.

  • Catalog — browse and manage every known Cutco item.
  • People — track what each collector owns, wants, or has sold.
  • Sets — see progress toward completing curated Cutco sets.
  • Sharpening / Cookware / Tasks — log usage and maintenance events.
  • Wishlist — set target prices and get alerted when MSRP drops.
  • Stats — visual charts of the full collection.
  • Use the Admin link to unlock catalog editing, syncing, and MSRP diffs.
{% elif page == 'msrp_diff' %}

Compares the stored MSRP prices in the database against live prices scraped from Cutco.com.

  • Click Run Diff to start a background scrape — progress updates live.
  • Enable Write to DB to save the updated prices after the diff completes.
  • Results show items where the price has increased, decreased, or gone missing.
  • A Discord summary can be sent if a webhook is configured.
  • Can also be run from the CLI via python msrp_diff.py.
{% elif page == 'variants' %}

Variants represent the different handle colors or editions available for an item.

  • If an item has no real colors yet, it uses Unknown / Unspecified as a fallback variant.
  • Add named color variants (e.g. "Classic Brown", "Camo") to track color-specific ownership.
  • Unicorn flags can be tracked separately at SKU/item level, edge/blade-type level, and variant/color level.
  • Ownership entries are linked to a specific variant, not just the item.
{% elif page == 'people' %}

Collectors are the people whose Cutco ownership is tracked in the vault.

  • Click a person's name to view their full collection.
  • The Owned count shows items with status "Owned" only.
  • Each person can have a shareable collection card generated from their collection page.
  • Add as many collectors as you like — the vault is designed for groups and families.
{% elif page == 'tasks_manage' %}

Manage the list of kitchen tasks used in the Knife Task Log.

  • Preset tasks are built in and cannot be deleted.
  • Custom tasks can be added freely and deleted if they have no logged usage.
  • Tasks with logged usage cannot be deleted — remove the log entries first if needed.
  • Task names are case-insensitive and must be unique.
{% endif %}
