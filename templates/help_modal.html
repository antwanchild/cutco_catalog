{% if page == 'catalog' %}

The catalog lists every Cutco item known to the vault.

  • Search by name or SKU using the search box.
  • Filter by category or show only 🦄 unicorn items.
  • Sort any column by clicking its header.
  • Click an item name to see its full detail page.
  • Admin users can add, edit, or delete items.
{% elif page == 'sets' %}

Sets are curated groupings of Cutco items (e.g. a knife block set).

  • Select a person from the dropdown to see their personal completion percentage for each set.
  • Click a set name to see which items are owned vs. missing.
  • Sets are created automatically during a Catalog Sync, or manually with the + New Set button.
{% elif page == 'set_detail' %}

Shows which items in this set are owned and which are still missing.

  • Select a person to filter completion to their collection.
  • Click + Wishlist on any missing item to add it to that person's wishlist.
  • Click 🎁 Share Gift List (when a person is selected) to generate a shareable link showing only the missing items — great for gifting.
{% elif page == 'collection' %}

Shows everything a person owns, wants, or has sold/traded.

  • Use the checkboxes to select multiple entries, then use the bulk action bar to change their status at once.
  • Missing Items lists catalog items this person doesn't have yet — click + Add to log them.
  • Variant Gaps shows items they own but are missing specific color variants of.
  • Click 🃏 Share Card to generate a public shareable link of their collection.
{% elif page == 'sharpening' %}

Tracks when each knife was last sharpened.

  • Knives past the threshold (default 180 days) are marked overdue.
  • Log a new sharpening event with the form — pick the knife, date, and method.
  • Admin users can send overdue reminders to Discord.
  • The threshold is configurable via the SHARPEN_THRESHOLD_DAYS environment variable.
{% elif page == 'bakeware' %}

Tracks baking sessions for Cutco bakeware and cookware.

  • Log what you made, a rating (1–5), and optional notes.
  • Items unused past the threshold (default 60 days) are flagged as stale.
  • The threshold is configurable via BAKEWARE_THRESHOLD_DAYS.
  • Admin users can send stale reminders to Discord.
{% elif page == 'tasks' %}

Log which knife you use for which kitchen task to track usage patterns.

  • Only owned knives appear in the log form.
  • The Usage Patterns table shows each knife's most-used task.
  • 10 preset tasks are included — add your own via Manage Tasks.
  • Custom tasks can be deleted if they have no logged usage.
{% elif page == 'wishlist' %}

Tracks items people want to own, optionally with a target price.

  • Set a target price on a wishlist entry — when the item's MSRP drops to or below that price, it's highlighted as a 🎯 hit.
  • Filter by person using the dropdown.
  • Admin users can trigger a price check that sends Discord alerts for any hits.
{% elif page == 'matrix' %}

A cross-tab view showing who owns what at a glance.

  • Rows are items, columns are people.
  • Each cell shows ownership status color-coded by badge.
  • Useful for spotting gaps or duplicates across your group.
{% elif page == 'stats' %}

Charts and summaries of the collection.

  • Filter to a specific person to see their individual stats.
  • Charts show owned items by category, handle color distribution, edge type breakdown, catalog coverage, and estimated collection value.
  • The Collectors table ranks people by item count and estimated value.
{% elif page == 'item_detail' %}

Full detail view for a single catalog item.

  • Left column — color variants, sets this item belongs to, and top tasks it's been used for.
  • Right column — who owns it, with quick-add buttons for people who don't.
  • Sharpening history and task log appear below if any events have been logged.
{% elif page == 'sync' %}

Scrapes Cutco.com to discover new items and sets.

  • New items are grouped by category — check the ones you want to import.
  • New sets are listed separately — check any you want to add.
  • Items in blocked categories (e.g. Accessories) are hidden by default.
  • Only items with SKUs not already in the database are shown.
{% elif page == 'import' %}

Bulk-import ownership data from a CSV or XLSX file.

  • Required columns: person, item_name or sku, status.
  • Optional columns: color, notes, is_unicorn.
  • A preview is shown before any data is written — you can deselect rows.
  • Existing ownership entries are skipped (no duplicates created).
{% endif %}
